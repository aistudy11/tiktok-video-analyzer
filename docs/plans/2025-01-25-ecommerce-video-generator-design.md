# 跨境电商视频生成器 - 设计文档

> 创建时间：2025-01-25
> 状态：已验证，待实施

## 一、产品定位

### 核心定位
**效率型工具** - 用户不想学习方法论，只想快速得到可用的结果

### Slogan
> "输入爆款链接，60秒生成同款脚本"
> "看到爆款 → 得到脚本 → 直接开拍"

### 差异化
- 竞品（FastMoss、Kalodata）：看数据（销量、达人、关键词）
- 我们：**快速出片**（分析→脚本→视频）

### 关键设计决策

1. **Level 2 输出** - 不只是脚本，要生成视频
2. **用户参与环节** - AI生成脚本初稿，用户可编辑调整，再生成视频
3. **个性化** - 用户可以根据自己的产品/风格修改脚本

## 四、技术方案

### 4.1 脚本生成

**输入**：AI分析结果（爆款原因、镜头语言、AI视频提示词）

**输出**：结构化脚本
```json
{
  "title": "脚本标题",
  "total_duration": "60秒",
  "scenes": [
    {
      "scene_number": 1,
      "duration": "0-3秒",
      "type": "hook",
      "narration": "开场话术...",
      "visual_description": "画面描述...",
      "notes": "拍摄建议..."
    },
    {
      "scene_number": 2,
      "duration": "3-15秒",
      "type": "problem",
      "narration": "痛点描述...",
      "visual_description": "画面描述...",
      "notes": "拍摄建议..."
    }
  ],
  "bgm_suggestion": "节奏感强的电子音乐",
  "cta": "关注+购买引导"
}
```

### 4.3 视频生成

**技术选型**：调用第三方API，不自建

| 决策 | 选择 | 理由 |
|------|------|------|
| 架构 | API调用 | 服务器压力转移给API服务商 |
| 主选API | Runway Gen-3 | 效果好，生态成熟 |
| 备选API | Pika、Kling | 价格/效果备选 |
| 数字人 | 后续Phase | HeyGen，非MVP范围 |

**流程**：
```
用户点击"生成视频"
    → 检查余额/扣费
    → 调用Runway API
    → 轮询等待结果
    → 返回视频URL
    → 用户下载
```

## 五、商业模式

### 定价策略

| 计划 | 价格 | 视频分析 | 视频生成 | 目标用户 |
|------|------|---------|---------|---------|
| **Free** | $0 | 3次/天 | 1次/月体验 | 体验用户 |
| **Basic** | $19/月 | 无限 | 20次/月 | 个人卖家 |
| **Pro** | $49/月 | 无限 | 100次/月 | 专业卖家 |
| **按量** | $1-2/次 | - | 不限 | 偶尔使用 |

### 成本考虑

- Runway API 成本约 $0.05/秒视频
- 60秒视频成本约 $3
- 需要后续市场调查验证定价是否能覆盖成本
- 可能需要根据实际成本调整价格
